SPONSTER

Sponster disrupts advertising by allowing a consumer to specify their interests and then rewarding them for consuming advertisements based on those interests. In order to get onto as many devices as quickly as possible the client and I chose to use the Ionic framework. This allowed us to provide hybrid native apps to those on mobile devices while reusing the codebase for those browsing on desktop.
Technologies I leveraged:
  • Ionic
  • Ruby on Rails
  • Grape (mounted to expose an API)

GOPRO CART AND CHECKOUT

GOPRO CART AND CHECKOUT

GoPro Cart handles all of GoPro online shopping. It is a fully customized web store built on the Netsuite platform. This is a responsive single page application built with Backbone.js and hosted in the Netsuite ecosystem. Since Netsuite is notoriously hard to develop for, I also developed several automated tools (mostly using Node.js) for automatically deploying code to Netsuite as well as mocking and stubbing their backend when possible.
Technologies I leveraged:
  • SASS/CSS
  • Backbone.js
  • Node.js
